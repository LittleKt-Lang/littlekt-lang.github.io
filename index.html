LittleKt 语言能力清单

基于 PyKt (Python 2 实现) 的测试验证，语言规范本身与宿主无关。

1. 基本语法结构

1.1 函数

  • 定义：fun name(参数列表): 返回类型 { 语句块 }
  • 参数类型注解：必须显式指定（如 n: Int）
  • 单表达式简写：fun add(a: Int, b: Int) = a + b
  • 默认参数：fun greet(name: String, greeting: String = "Hello")
  • 泛型函数：fun <T> identity(x: T): T

1.2 类

  • 定义：class Name(val prop: Type, var prop: Type) { ... }
  • 构造参数必须带 val/var 及类型注解
  • 支持成员函数和属性
  • 泛型类：class Box<T>(val value: T)

1.3 变量

  • 不可变：val
  • 可变：var
  • 类型推断（根据初始化表达式推导）

1.4 控制流

  • if 表达式：可返回值的 if，支持 else if 链
    • 所有分支类型必须一致
  • 可用于赋值、返回、函数参数等任意表达式上下文

2. 类型系统

2.1 内置类型

  • Int, String, Boolean, Double, Any（根类型）

2.2 泛型

  • 泛型函数：fun <T> foo(x: T): T
  • 泛型类：class Box<T>(val value: T)
  • 泛型方法：成员函数可声明自己的类型参数 <U>
  • 类型推断：调用时可不指定类型参数（如 Box(100) 推断为 Box<Int>）
  • 定义时类型检查：返回类型与表达式类型不匹配时报错
  • 嵌套泛型（如 List<List<Int>>） 解析可能受限

2.3 类型注解

  • 强制显式注解：参数、类属性、返回值（除非支持推断的局部变量 val x = 10）

3. Lambda 表达式与高阶函数

3.1 Lambda 语法

  • 多参数形式：{ x, y -> x + y }（参数无需显式类型注解）
  • 单参数隐式 it：{ it * it }
  • 闭包：捕获外部变量（包括可变 var）
  • 参数显式类型注解：{ x: Int -> x } 未验证通过（可能不支持）

3.2 高阶函数

  • Lambda 作为函数参数：fun operate(a: Int, b: Int, op: (Int, Int) -> Int): Int
  • Lambda 作为返回值：fun multiplier(factor: Int): (Int) -> Int = { x -> x * factor }

3.3 集合高阶函数（标准库）

  • List.map, List.filter, List.filterNotNull
  • Map.mapKeys（可能还有 mapValues）

4. 标准库（内置集合与操作）

4.1 数组（Array）

  • 创建：arrayOf(vararg items)、arrayOfNulls(size)
  • 属性：size
  • 索引访问：arr.get(index) 或 arr[index]

4.2 列表（List）

  • 创建：listOf(vararg items)
  • 属性：size, isEmpty
  • 方法：get(index), contains(item), indexOf(item)
  • 可变操作：add(item), removeAt(index)

4.3 映射（Map）

  • 创建：mapOf(pair1, pair2, ...)，pair 由 to 中缀创建（如 "a" to 1）
  • 字面量：["key" to value, ...]
  • 属性：size, keys, values
  • 方法：get(key), containsKey(key), put(key, value)（可变）

4.4 字符串（String）

  • trim(), toUpperCase(), toLowerCase()
  • 属性：length
  • substring(start, end), startsWith(prefix), endsWith(suffix), contains(sub), replace(old, new)
  • 转换：toIntOrNull(), toDoubleOrNull()

4.5 整数（Int）

  • toString（作为属性调用）

5. 语法糖与便捷特性

5.1 字符串模板

  • 简单变量："Hello, $name"
  • 表达式："Sum: ${a + b}"（基础支持）

5.2 注释

  • 行注释：// ...
  • 块注释：/* ... */

6. 已知限制（语言层面未实现/待定）

  • Lambda 参数的显式类型注解（如 { x: Int -> x }）
  • 嵌套泛型类型（如 List<List<Int>>） 解析可能受限
  • when 表达式
  • 数据类（data class）
  • 解构声明
  • 扩展函数/属性
  • 协程 / 挂起函数
  • 密封类 / 对象声明 / 单例

说明：✅ 支持 — 已通过测试验证；⚠️ 受限 — 已知部分场景不支持；❌ 不支持 — 测试失败或未实现；❓ 未测试 — 尚未验证，可能支持或缺失。

语言定位：LittleKt 是与宿主无关的通用语言，以上能力基于 PyKt (Python 2 实现) 的测试确认，未来其他实现（如 LittleKt-JVM、LittleKt-JS）应遵循相同的语言规范。

版本信息：本文档基于 2026 年 7 月测试结果编写。
